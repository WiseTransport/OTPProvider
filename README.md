## How to Use

### First time

If starting for the first time, run `download_otp.sh`

### Build the graph

```bash
java -Xmx2G -jar otp-2.6.0-shaded.jar --build --save resources
```

### Launch

Run from the root directory.

Windows
```bash
start.bat
```
\
Linux
```bash
chmod +x start.sh && ./start.sh
```
